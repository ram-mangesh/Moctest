# Moctest Repository

This repository contains multiple components:

- **Mocktest**: A Flutter application (see `Mocktest/README.md`).
- **examprep_flutter**: Another Flutter project with various dependencies.

## Overview

The `Mocktest` folder includes a Spring Boot backend (`mock/pom.xml`) and a React frontend (under `mock/src`).

The repository is set up for development on Windows.

## Getting Started

### Prerequisites
- Java 17
- Flutter SDK
- Node.js (for the React part)
- Git

### Build and Run
1. Clone the repo.
2. Navigate to the desired subproject (e.g., `Mocktest`).
3. Follow the specific README in each subfolder for detailed instructions.
